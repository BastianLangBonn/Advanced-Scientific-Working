\frametitle{Scientific problems faced in R \& D}
Real world optimization of energy efficient vehicle control
\begin{itemize}
	\item Transfer controller to real device
	\item Evaluate controller on real device
	\item Cross reality gap (if controller does not transform well)
	\item Compare results with other energy efficient tasks
	\item Create (more elaborate) model
	\item Use multi objective approach
\end{itemize}
\end{frame}

\begin{itemize}
	\item Literature is proof of concept, but approach has to be specified for the used device
	\item Approach needs a good model
	\item Requires heavy adjusting of the used EA 
	\item Has never been applied to a similar device
\end{itemize}
